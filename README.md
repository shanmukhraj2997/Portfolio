# Premium Portfolio Website

A modern, high-performance portfolio website featuring sleek Glassmorphism design, dynamic animations, and a responsive projects slider.

**Live Demo**: [shanmukhraj2997.github.io/Portfolio/](https://shanmukhraj2997.github.io/Portfolio/)

## ✨ Key Features

- 💎 **Glassmorphism UI**: High-end aesthetic with frosted-glass effects, subtle borders, and backdrop filters.
- 🌓 **Intelligent Theme Toggle**: Seamless transitions between Light and Dark modes with optimized contrast.
- 🎢 **Horizontal Projects Slider**: Interactive project showcase with side-scrolling and smart navigation arrows.
- 🌊 **Dynamic Background**: Beautiful mesh gradient background that provides a premium look.
- 🚀 **Performance Optimized**: Built with Vanilla JS and CSS for maximum speed and smooth interaction.
- 📏 **Sidebar Stabilization**: Robust flexbox layout that ensures sidebar visibility on all desktop sizes.
- 📜 **Scroll Reveal**: Sections animate into view using the Intersection Observer API.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **Tailwind CSS**: Utility-first CSS for rapid, modern styling.
- **Vanilla JavaScript**: Dynamic content injection and interactive features.
- **Font Awesome**: Premium vector icons for professional visuals.
- **Google Fonts (Inter)**: Clean, modern typography.

## 📁 Code Structure

The project follows a clean separation of concerns:
- `Portfolio1.html`: Structured content and layout.
- `styles.css`: All premium styling, animations, and theme variables.
- `script.js`: Data-driven content injection and interaction logic.
